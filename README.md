# TentLait website

#### `JavaScript / ReactJS / TypeScript / NextJS`

# [https://tenlait.lviv.ua/](https://tenlait.lviv.ua/)

## Getting Started

```bash
yarn install

yarn dev

yarn build

yarn deploy
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the
result.

## Stack

- Next.js
- TypeScript

Also use:

- Scss
- ftp-deploy
